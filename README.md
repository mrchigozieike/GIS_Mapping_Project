# GIS Mapping Project

## Overview
This project is a Geographic Information System (GIS) web application that displays important locations in Nigeria on an interactive map using the Leaflet mapping library. Users can explore different landmarks such as parks, museums, and universities across Nigeria.

## Features
- Displays **20+ markers** across Nigeria
- Click on any marker to see a **popup with location information**
- **Filter system** to display only parks, museums, or universities
- **Interactive map** with zoom and pan functionality
- Built using a lightweight open-source mapping library

## Technologies Used
- HTML
- CSS
- JavaScript
- Leaflet.js (mapping library)
- OpenStreetMap (map tiles)
